# Titan Motors - Professional Automobile Website

A high-end, premium website developed for Titan Motors, featuring a sleek dark-themed design and comprehensive automotive features.

## Tech Stack
- **Framework**: React (Vite)
- **Styling**: Vanilla CSS (Custom Design System)
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Key Features
- **360° Vehicle View**: Interactive simulated rotation for flagship models.
- **EMI Calculator**: Integrated loan and installment calculation tool.
- **Inventory Management**: Filterable vehicle listing with detailed specs.
- **Booking Systems**: Functional flows for Test Drives and Service maintenance.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Glassmorphism UI**: Modern, premium aesthetic with subtle blurs and gradients.
